# Spring Boot Learning Project

This repository contains a Spring Boot project created as part of my learning journey in backend development with Java. The project focuses on understanding Spring Boot fundamentals, project structure, dependency management, and building simple REST controllers.

## Project Overview

Key concepts and features covered in this project include:

- Spring Boot project setup and configuration
- Understanding project structure
- Dependency management with Maven
- Building REST controllers
- Running and debugging Spring Boot applications
- Automatic application restarts using DevTools
- Configuring application properties
- Dependency Injection (constructor and setter)
- Spring IoC container
- Configuring beans using annotations

This project is intended for **educational purposes** to solidify foundational Spring Boot skills.

## Technology Stack

- **Language:** Java 17  
- **Framework:** Spring Boot  
- **Build Tool:** Maven  
- **IDE:** IntelliJ IDEA
